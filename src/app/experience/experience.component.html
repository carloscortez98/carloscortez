Login / Signup

Usando MongoDB, ExpressJs, Angular y NodeJs - Finalizado

  Aplicación de una sola página. Posee un sistema de registro de usuarios y con la capacidad de iniciar sesion con los datos almacenados. Es posible eliminar la cuenta y, a su vez, perder el ingreso al sitio. Autenticacion del usuario por medio de JWT. Validaciones de formularios realizadas desde Angular y Express. Información almacenada en MongoDB Atlas. Diseño con CSS, jQuery y Angular Material como principal proveedor. Backend alojado en Heroku, frontend disponible gracias a Netlify.

Single Page Application para emprendimiento de ropa

Usando MongoDB, ExpressJs, Angular y NodeJs - Finalizado

  Acceso para todo público. Sistema de filtros (precio menor/mayor, ofertas, colores). Usuario único (admin), el cual tiene acceso a un Panel de control desde donde se puede crear, editar o eliminar mercadería. Autenticacion del usuario administrador por medio de JWT. Validaciones de formularios realizadas desde Angular y Express. Información almacenada en MongoDB Atlas e imágenes en el servidor. Diseñado con Angular Material y jQuery. Adaptado para dispositivos móviles y tablet. Actualmente no se encuentra disponible en internet.

Control de stock de productos

Usando MongoDB, ExpressJs, Angular y NodeJs - Finalizado

  Mi primer proyecto personal. Aplicación de una sola página. Contiene un usuario único (admin) que permite el acceso completo al sitio, además es capaz de crear, modificar y eliminar productos de la base de datos. Incluye el registro de cada producto editado, señalando también la fecha en la cual se realizó cada cambio. Validaciones de formularios realizadas desde Angular y Express. Información almacenada en MongoDB Atlas e imágenes en el servidor. Diseño con CSS, jQuery y Angular Material como principal proveedor. Backend alojado en Heroku, frontend disponible gracias a Netlify.

E-commerce de productos

Usando MySQL y PHP en Codeigniter - Finalizado

  Proyecto entregado como examen de final de curso. Cuenta con sistema de SignUp/Login de usuarios además de tener un usuario ADMIN, el cual es el único capaz de crear, editar o eliminar productos de la base de datos. Validaciones de formularios realizadas con Javascript y PHP. Desarrollado utilizando Bootstrap en su mayoría, lo que hace al sitio adaptable a distintos dispositivos, también contiene partes creadas con jQuery. No incluye integración de pagos. Disponible en GitHub.
